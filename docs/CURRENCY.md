# สกุลเงินแสดงผล (THB default)

- **ระบบบัญชี / wallet / ราคาสินค้า** เก็บและเรียกเก็บเป็น **บาท (THB)** เสมอ
- **แสดงผลเป็นสกุลอื่น** ได้ในหน้าลูกค้า — ใช้อัตราแลกเปลี่ยนจาก **Frankfurter** (`https://api.frankfurter.dev`) ซึ่งอิง **ECB** ไม่ต้องใช้ API key
- Backend โหลด `GET /v1/latest?from=THB` และ **cache ~1 ชม.** เปิดเผยผ่าน `GET /api/v1/currency/rates`
- การแปลงบนหน้าจอ: `จำนวนในสกุลเป้าหมาย = จำนวน THB × rates[สกุล]` (Frankfurter ให้ `rates` เป็น “1 THB = เท่าไรของสกุลนั้น”)

> อัตราเป็นข้อมูลอ้างอิง ไม่ realtime — ไม่เปลี่ยนยอดเงินจริงในระบบ
